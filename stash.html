Git Stash Cheat Sheet

Use git stash when you donâ€™t want to commit yet but need to temporarily save your incomplete code.

Save Changes

git stash save "WIP: changes before switching branches"

View Stashed Changes

git stash list

Apply Stash (Without Removing)

git stash apply
git stash apply stash@{1}

Apply Stash (And Remove It)

git stash pop
git stash pop stash@{1}

Remove a Specific Stash

git stash drop stash@{0}

Clear All Stashes

git stash clear

Tip

git stash apply keeps the stash, while git stash pop removes it after applying.
